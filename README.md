# 🧠 밈랭 (MemeLang)

밈랭(MemeLang) 은  
한글로 작성하는 프로그래밍 언어다.

---

## ✨ 특징

- 🇰🇷 100% 한글 문법
- 🎮 변수, 반복문, 조건문 지원
- 난해함(난해한 프로그래밍 언어)
  
---

## 🧩 기본 개념

### 📦 상태 슬롯 (변수)
밈랭에는 3개의 기본 슬롯이 있음.

| 슬롯  |    설명    |
| 주머니| 상태 슬롯 1 |
| 가방  | 상태 슬롯 2 |
|  손   | 상태 슬롯 3 |

슬롯은 숫자 값을 가짐.

---

## 📘 문법 요약

### ▶ 슬롯 선택
주머니 집다

### ▶ 값 증가 / 감소
집는다
놓는다

---

### ▶ 출력

말한다 주머니
말한다 안녕하세요


---

### ▶ 입력


묻는다 숫자를_입력하세요


---

### ▶ 조건문


확인한다
말한다 0입니다
아니인가?>3
말한다 3보다_큽니다
아니다
말한다 기타
끝확인


조건은 **선택된 슬롯 값**을 기준으로 판단.

지원 연산자:
- `>`
- `<`
- `==`
- `!=`

---

### ▶ 반복문


다섯 번
집는다
다시한다


또는 슬롯 값 사용:


손 번
